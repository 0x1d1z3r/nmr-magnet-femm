# nmr-magnet-femm
Software to calculate magnetic field homogeneity in DIY NMR magnets

Inspired by https://doi.org/10.21203/rs.3.rs-3303984/v1

## Requirements:
MATLAB (tested on R2024b) / Octave (not yet tested)

FEMM: https://www.femm.info/wiki/Download
